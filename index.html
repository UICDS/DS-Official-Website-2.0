WHAT IS DATA SCIENCE


Some academic terms about data science.


DATABASE

Being an organized set, a database is a collection of data, usually stored and accessed from a computer system.

MACHINE LEARNING

Machine learning studies mathematic models and algorithms, which computer systems use to progressively improve their performances on specific tasks.

DATA MINING

Data mining means discovering patterns in large data sets, by collecting information from databases. It is a kind of data analysis technique.

CLOUD COMPUTING

Cloud computing is a kind of new computing model, which uses shared pools of configurable computer system resources and higher-level services, often over the Internet.

DATA ANALYSIS

To discover useful information, inform conclusions and help make decisions, we need to inspect, cleanse, transform and model data. That is the process of data analysis.

DATA ARCHITECTURE

Data architecture is composed of models, rules, standards or policies that govern controls collecting, storing, arranging, integrating and using data.
